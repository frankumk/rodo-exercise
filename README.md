# rodo-exercise

## Project setup
```
npm install
```

### Serves api with json-server
```
npm run api
```

### Starts Server
```
npm run start
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Form work is in @src/components/SearchForm.vue
### API work is in @server/server.js
